# GameClient

This directory is reserved for the Unity project of Bike Super Racing.

## Current baseline state
This repository-safe baseline includes:
- canonical folder structure references;
- `.asmdef` files;
- domain model skeletons;
- service contracts;
- baseline runtime stubs.

## Important
Real Unity assets must be created in Unity Editor by the Unity programmer:
- `.unity` scenes;
- `.asset` ScriptableObject instances;
- prefabs;
- inspector bindings;
- project settings and package setup.

The authoritative naming and structure rules are defined in:
- `Docs/architecture/naming-convention.md`
- `Docs/architecture/project-structure.md`
- `Docs/architecture/architecture-baseline.md`
