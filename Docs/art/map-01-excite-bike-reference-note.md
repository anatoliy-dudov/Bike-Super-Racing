# Bike Super Racing — `map_01` Excite Bike Reference Note

## Purpose
This note records the visual reference direction for `map_01` after the decision to use Excite Bike (Nintendo) as a stronger track reference.

The goal is not to copy the original track literally.
The goal is to inherit the core readability qualities that make the track profile instantly understandable.

## Key reference takeaways
For `map_01`, the most useful Excite Bike track qualities are:
- very readable ground profile;
- clear bright top edge of the rideable surface;
- simple dirt mass with limited internal texture;
- obvious humps, ramps and crests;
- strong separation between track, grass strip and sky;
- low decorative noise.

## What this means for Bike Super Racing
`map_01` should move closer to:
- simpler track silhouette;
- clearer surface lip;
- more rhythmic arcade humps;
- cleaner dirt texture;
- more confident ramp readability.

## What should not be copied blindly
The project should not inherit:
- unnecessary retro limitations that hurt modern clarity;
- over-flat sameness if it weakens the intended control-feel validation;
- exact visual imitation instead of adapted inspiration.

## Practical art implications
The next pass on `map_01` should improve:
- surface edge readability;
- ramp readability before contact;
- crest rhythm;
- internal dirt noise reduction;
- stronger simple arcade track identity.

## Decision rule
If a track detail makes the scene more decorative but less instantly readable, it should be removed.

The Excite Bike reference is being used mainly as a readability benchmark, not as a nostalgia skin.
