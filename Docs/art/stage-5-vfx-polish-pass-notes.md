# Bike Super Racing — Stage 5 VFX Polish Pass Notes

## Purpose
This note records the goals of the focused polish pass for the Stage 5 VFX must-have set.

The polish pass stays intentionally narrow.
It improves only:
- start dust direction and shape clarity;
- run dust subtlety and cleanliness;
- finish burst readability and compactness.

It does not expand the VFX scope.

## Why this pass exists
After the first Stage 5 draft, the VFX direction was correct, but the following areas still had room for improvement:
- start dust could feel more directional and impulse-driven;
- run dust could be slightly cleaner and quieter;
- finish burst could feel more arcade-clean and less like a generic sprite star.

## Goals of the polish pass
The polish pass must improve:
- start-state impulse readability;
- movement-state support without visual dirt buildup;
- finish confirmation clarity;
- overall VFX cleanliness at gameplay scale.

The polish pass must preserve:
- short effect lifetime;
- bike and track as visual priorities;
- low-noise arcade art direction;
- compatibility with the current first playable slice.

## What must not happen
The polish pass must not introduce:
- bigger dust clouds;
- lingering smoke-like masses;
- finish burst spectacle that blocks the rider;
- more VFX variety unrelated to the must-have set.

## Review checklist
After this pass, review:
1. does start dust feel more directional and energetic;
2. does run dust stay subtle and readable;
3. does finish burst confirm the finish without dominating the frame;
4. do all effects remain clearly secondary to bike and track;
5. is gameplay readability preserved in all four states.

## Intended decision after this pass
If the answers above are strong, the Stage 5 VFX set should be accepted as the VFX baseline for the current first playable milestone and the team should move to an integration check of the full visual slice.
